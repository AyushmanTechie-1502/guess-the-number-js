# Guess The Number – JavaScript

A small JavaScript game built to practice DOM manipulation and basic game logic.

## Features
- Random number generation (1–10)
- Score and high score tracking
- UI updates for win and loss states
- Reset game without page reload

## Tech Used
- HTML
- CSS
- JavaScript (Vanilla)

## What I learned
- Handling user input with events
- Updating the DOM dynamically
- Managing state in a small JS application
